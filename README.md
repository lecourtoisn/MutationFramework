## Contents

  Nous allons développer un framework de tests par mutations.
  A partir d'un programme donné, nous allons créer plusieurs programmes mutants et leur appliquer un ensemble de tests. Le but étant d'avoir un maximum de mutants ne passant pas les test, afin de confirmer que les tests sont pertinents et donc de valider le code d'origine.
